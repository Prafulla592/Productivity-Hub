# Productivity Hub

Productivity Hub is an AI-powered career planning platform that helps users:

- Assess current skills and background
- Get tailored career recommendations
- Identify skill gaps for target roles
- Build practical learning roadmaps

## Tech Stack

- React + TypeScript + Vite (frontend)
- Express + TypeScript (backend)
- Tailwind CSS + shadcn/ui (UI)
- Drizzle ORM + PostgreSQL (data)

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Add environment variables in `.env`.

3. Start development server:

```bash
npm run dev
```
